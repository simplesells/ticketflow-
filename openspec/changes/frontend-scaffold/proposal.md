## Why

需要前端项目基础和列表页面来展示后端数据。完成此步后浏览器能看到工单列表。

## What Changes

- 初始化 Vite + React 18 + TypeScript 前端项目
- 实现 HTTP 客户端 (fetch wrapper，调用后端 API)
- 实现路由配置 (React Router v6)
- 工单列表页面 — 从后端拉取数据以表格展示
- 按状态/优先级筛选功能

## Capabilities

### New Capabilities
- `frontend-base`: 前端项目骨架、路由、HTTP 客户端
- `workorder-list-ui`: 工单列表展示及筛选界面

### Modified Capabilities

## Impact

- 新增 `frontend/` 目录及 Vite + React 项目
